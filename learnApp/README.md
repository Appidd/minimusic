# 星梦园音乐考研

这是一款针对音乐考研的刷题小程序，里面涉及到考试和刷题练习，其分为一下几个功能模块：

## 功能模块介绍
### 刷题考试
- 单选题：出题方式有文本、图片、音频；
- 多选题：出题方式有文本、图片、音频；
- 简答题：出题方式有文本、图片、音频；
- 判断题：出题方式有文本、图片、音频；
- 填空题：出题方式有文本、图片、音频；

### 用户管理
- 登陆注册：用户基本信息管理；
- 用户权限：用户权限管理；

### 课程管理
- 视频课程：不同用户根据权限展示不同课程；


## 开发技术栈介绍

### 前端
- 小程序端：uniapp（vue、ts、vuex）、ts、css；
- 后端管理：vue3(setup语法糖、vuex、vue-router等插件)、ts、css；

### 后端
- 数据库：myslq关系型数据库
- 后端语言：php
